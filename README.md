# Название: clicker

# Какую проблему решает проект  
Предоставляет пользователю простое и увлекательное времяпрепровождение для снятия стресса, а также демонстрирует навыки разработки GUI-приложений на Python.

# Целевая аудитория 
Любители казуальных игр, фанаты жанра incremental/idle, люди всех возрастов, ищущие расслабляющий игровой процесс.

# Цели и задачи  
- Создать игровое приложение с механикой накопления ресурсов через клики и улучшения.  
- Реализовать интуитивный интерфейс, систему апгрейдов, сохранение прогресса.  
- Обеспечить визуальную и звуковую обратную связь для вовлечения игрока.

# Функционал  
- Клик по кнопке для получения ресурсов.  
- Счётчик ресурсов с мгновенным обновлением.  
- Магазин улучшений (автокликеры, множители дохода).  
- Пассивный доход от купленных улучшений.  
- Система достижений.  
- Сохранение и загрузка прогресса.  
- Визуальные и звуковые эффекты (анимация клика, звуки покупок).
